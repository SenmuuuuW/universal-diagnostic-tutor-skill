# 准备度门槛：通过与暂不通过

准备度取决于可见证据，不取决于概念是否已经讲过。下面两个案例使用同一输出
结构，但不会给学习者制造永久标签。

## 案例一：可以推进

**当前概念：** 条件概率中的条件样本空间。

**观察到的证据：** 学习者独立算出

\[
P(A\mid B)=\frac{P(A\cap B)}{P(B)},
\]

并解释“分母换成 \(P(B)\)，因为现在只在 \(B\) 已发生的情况里比较”。在一个
改变数字和语境的近迁移题中，仍能选对分母。

**准备度决定：推进。** 正确答案、正确理由和近迁移成功形成了相互支持的证据，
可以进入贝叶斯公式的基础应用。

**下一步：** 给一题只包含两个事件的贝叶斯公式题，不立即跳到多类别或连续分布。

**状态更新：** 条件样本空间已练习、已检查、已确认；贝叶斯公式仍未确认。

## 案例二：先复习

**当前概念：** 导数表示瞬时变化率。

**观察到的证据：** 学习者正确说出 \(x^2\) 的导数是 \(2x\)，但表示“只是记得
公式”，无法解释导数与斜率或变化率的关系，也没有做过近迁移。

**准备度决定：先复习。** 一次公式回忆只支持程序识别，不能证明概念理解，更
不能支持直接进入链式法则。

**下一步：** 用一个位置随时间变化的小例子重建“某一时刻变化有多快”，随后只
问一道解释题。

**状态更新：** 幂函数求导已练习但理由较弱；瞬时变化率仍未确认；下一检查点是
能否用自己的话解释导数。

## 关键区别

第一个案例同时有解释和近迁移证据；第二个案例只有一次正确结果。Tutor 因此做出
不同决定，而不是把所有正确答案都当作进阶通行证。
