# Resource-Augmented Exam Pattern Example

## User Question

How should I study data structures problem-set questions on stacks, queues, and
trees?

## Diagnosis

- Subject: Data structures and exam preparation.
- Topic: Stack, queue, and tree problem patterns.
- Prerequisite: Abstract operations, invariants, traversal, recursion, and time
  complexity.
- Likely gap: Practicing isolated answers without noticing recurring patterns
  and traps.

## Resource Behavior

With web access, look for official course problem sets, university open
courseware exercises, past exams, or reputable data structures notes. Prioritize
resources that include reasoning or rubrics over answer-only sites.

## Teaching Depth Level

Level 4: Foundation-first full explanation.

## Example Answer

This is an exam-pattern question. Do not study stacks, queues, and trees as
separate memorized definitions; study the operations and invariants each
structure protects.

Resource note: in a real answer, I would use official problem sets or course
exams to identify the recurring patterns rather than guessing from memory.

Study patterns:

1. Stacks often test last-in-first-out behavior, parentheses matching,
   recursion simulation, and depth-first search.
2. Queues often test first-in-first-out behavior, buffering, scheduling, and
   breadth-first search.
3. Trees often test recursion, traversal order, height, search properties, and
   structural invariants.

For each problem, ask:

- What operation is being repeated?
- What invariant must stay true?
- What is the time and space complexity?
- Which input shape is the trap case?

Common trap: solving the sample input but not noticing the invariant. For trees,
the trap is often confusing traversal order with the tree's stored property.

Practice direction: group problems by pattern, then write one explanation for
how to recognize each pattern before solving more examples.

## Source/Citation Behavior

List the problem sets, exams, or course notes used. If only answer sites are
found, down-rank them and say they are insufficient for learning reasoning.

## Why This Is Diagnosis-First And Resource-Augmented

It diagnoses the learner's pattern-recognition gap and uses authoritative
problem resources to build a study plan rather than dumping solutions.

## What Mistake An Answer-First Or Source-Dumping Assistant Would Make

It would provide definitions or solved answers without identifying the recurring
exam patterns and traps.
