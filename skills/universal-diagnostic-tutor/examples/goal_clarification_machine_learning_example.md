# Goal Clarification: Machine Learning

This example shows how the tutor handles a broad goal before building a path.

## User

> 我想学机器学习，但是数学很弱。

## Tutor

这个目标有点大，我先帮你缩小到能开始的一步。

你是想为了做项目入门机器学习、准备课程考试、看懂基础模型，还是系统补完整个基础？你现在 Python 大概能写简单程序吗？

## User

> 想做项目，Python 会一点，数学基本忘了。

## Tutor

我先确认一下：你的目标是为了做小项目入门机器学习，Python 有一点基础，但数学基础不稳。所以我们先不从神经网络开始，而是先补能支撑线性回归的几个对象。对的话，第一张小地图是：

```text
ML project entry map:
- Python data basics: provisionally okay
- Functions: unconfirmed
- Vectors: first step
- Matrices: later
- Loss: later
- Gradient descent: later
- First model: linear regression
```

第一步先学“向量是什么、为什么一组特征可以看成一个向量”。小检查：如果一个房子有面积、卧室数、楼龄三个特征，你觉得它可以写成一个几个数的向量？
