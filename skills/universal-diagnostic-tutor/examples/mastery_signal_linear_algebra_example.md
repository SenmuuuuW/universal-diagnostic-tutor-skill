# Mastery Signal Linear Algebra Example

## Scenario

The tutor asked: "If \(u=[3,6]\) and \(v=[1,2]\), why are they scalar multiples?"

The learner answers: "Because \(3\) is bigger than \(1\), and \(6\) is bigger
than \(2\), so they point the same way."

## Expected Tutor Behavior

- Interpret the answer as partially correct.
- Preserve the correct intuition that component ratios matter.
- Repair the missing concept: the same scalar must work for all components.
- Do not restart from vectors from zero.
- Ask one focused check.

## Example Tutor Response

你抓到了一部分：我们确实要比较两个向量的分量关系。

但“都变大”还不够。真正的关键是：**每个分量都要乘同一个数**。

这里 \([1,2]\) 乘以 \(3\) 会得到：

\[
3[1,2]=[3,6]
\]

所以 \([3,6]\) 和 \([1,2]\) 是标量倍数，不只是因为两个分量都变大，而是因为同一个倍数 \(3\) 同时适用于两个分量。

小检查：\([2,6]\) 是 \([1,2]\) 的标量倍数吗？你只要看两个分量是否对应同一个倍数。

## Why This Example Works

The tutor treats the learner's answer as a mastery signal: some structure is
present, but the precise scalar condition is missing. It repairs only that gap.
