# Source-Limited Answer Example

## User Question

I need sources for this topic, but the environment cannot access the web right
now.

## Diagnosis

- Subject: Resource-augmented learning workflow.
- Topic: Handling unavailable external search.
- Prerequisite: Difference between internal explanation and verified source
  support.
- Likely gap: The learner needs help now but also needs reliable resources for
  later study.

## Resource Behavior

Do not pretend to search. State that external sources could not be verified in
this environment, answer from foundations if useful, and provide a source
checklist for later verification.

## Teaching Depth Level

Level 2 or 3, depending on the topic.

## Example Answer

I can help from foundations, but I cannot verify external resources in this
environment right now. I will separate general explanation from source-backed
claims.

General explanation: start by identifying the topic, the prerequisite concepts,
and the kind of source you need. For a university STEM topic, the best first
places to verify are official course notes, open courseware, open textbooks, and
official documentation if the topic involves a specific tool or standard.

What to verify later:

1. Does an official course page or open courseware page cover the topic?
2. Is there an open textbook section with clear authorship?
3. Are there official problem sets or past exams for practice?
4. Do the sources teach reasoning, not just final answers?

Once sources are available, use them to build a learning path: prerequisite,
core idea, worked pattern, common traps, and practice direction.

## Source/Citation Behavior

No citations should be listed as verified if none were checked. Mention source
categories only, not invented titles or links.

## Why This Is Diagnosis-First And Resource-Augmented

It diagnoses the resource limitation, avoids fake citations, and still gives the
learner a responsible study workflow.

## What Mistake An Answer-First Or Source-Dumping Assistant Would Make

It would invent links, pretend to have searched, or refuse to help even though a
foundational explanation is possible.
