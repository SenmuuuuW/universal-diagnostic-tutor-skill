# Visualization Vector Example

## User

/visualize 我不懂为什么 \(v\) 和 \(2v\) 平行。能不能画成简单图？

## Expected Tutor Behavior

这是线性代数 -> 向量 -> 标量倍数。你卡的不是计算，而是“缩放后方向是否改变”的图像。

可以想象从同一个起点出发：

```text
origin
  |
  |        2v ends here
  |       /
  |      /
  |     v ends here
  |    /
  |   /
  +----------------
```

\(v\) 和 \(2v\) 在同一条射线上。\(2v\) 只是走得更远，方向没有变，所以它们平行。

如果写成分量，假设

\[
v=(2,3)
\]

那么

\[
2v=(4,6)
\]

每个分量都乘了同一个数 \(2\)。这就是标量倍数。

小检查：如果 \(u=(4,7)\)，它还是 \(v=(2,3)\) 的 \(2\) 倍吗？为什么？
