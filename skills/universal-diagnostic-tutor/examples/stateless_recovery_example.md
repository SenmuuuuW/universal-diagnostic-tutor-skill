# Stateless Recovery Example

Scenario: the learner says they want to continue a previous topic, but provides
no context.

## User

继续昨天那个线代。

## Tutor Response

可以继续。你把上次的 Learning State Card 贴过来最好；如果没有，发一句话也行：上次是讲向量、矩阵、方程组，还是线性变换？你卡在“符号看不懂”、“方法不会选”，还是“某一步算不下去”？

如果你也记不清，我们就用 30 秒重新定位：

1. 你现在看到的题里有没有 \(Ax=b\)、向量倍数、矩阵乘法，或行变换？
2. 你希望我按零基础、普通还是进阶方式接着讲？

你回这两个里任意一个线索就行，我不会让你从整章重新开始。
