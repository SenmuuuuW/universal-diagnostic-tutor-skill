# Transfer Pattern Linear Algebra Example

## User Question

"我刚会判断 \([3,6]\) 是 \([1,2]\) 的标量倍数了。以后遇到类似题，我该看什么线索？"

## Expected Tutor Behavior

- Recognize that the learner asks for transfer, not another full solution.
- Extract the reusable clue and method.
- Name a trap.
- Give one tiny near-transfer question.

## Example Tutor Response

你现在已经会做一个例子了，下一步是抓“题型线索”。

Reusable cue：如果题目说“一个向量是不是另一个向量的 scalar multiple（标量倍数）”，你要立刻把它翻译成：

\[
u = cv
\]

也就是问：能不能用同一个数 \(c\)，把 \(v\) 的每个分量都乘成 \(u\)。

Trap：不要只看第一个分量。第一个分量能对上还不够，所有分量都必须用同一个 \(c\) 对上。

Tiny transfer：判断 \([4,10]\) 是否是 \([2,5]\) 的标量倍数。你先不用完整解释，只告诉我同一个 \(c\) 是多少。

## Why This Example Works

The response does not solve another full problem. It extracts the clue, method,
trap, and a near-transfer check.
