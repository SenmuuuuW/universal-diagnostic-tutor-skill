# Topic Scan Linear Algebra Resources Example

## User

我总是不懂为什么两个向量平行要写成 \(u=cv\)。如果我要补基础，有什么靠谱资料方向？

## Expected Tutor Behavior

Topic scan: 线性代数 -> 向量 -> 平行关系 / 标量倍数。前置概念是向量分量和“同一个标量倍数”。

先不急着算。两个向量平行不是说每个分量相等，而是说一个向量可以由另一个向量整体缩放得到：

\[
u=cv
\]

关键是同一个 \(c\) 要同时适配每个分量。比如 \(v=(2,3)\)，那么 \(2v=(4,6)\)，方向没有变，只是长度变了。

如果你要补资料，优先找两类：

- 大学线性代数课程里的 vector / scalar multiple / span 小节，用来补正式定义和练习。
- 可视化线性代数解释，用来理解“方向不变、长度缩放”的直觉。

如果当前环境能搜索，再去验证具体课程页或开放教材链接；如果不能搜索，就不要编造链接。

小检查：如果 \(v=(1,2)\)，\(u=(3,6)\)，它们是否满足 \(u=cv\)？
