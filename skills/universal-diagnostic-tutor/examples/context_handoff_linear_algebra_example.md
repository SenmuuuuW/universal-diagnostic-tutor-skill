# Context Handoff: Linear Algebra Example

Scenario: the learner pastes a Learning State Card in a new chat.

## User

Learning State Card:

- **Subject:** Linear algebra
- **Topic:** Scalar-multiple vectors
- **Current learning mode:** Standard Mode
- **Already understood:** A vector has components; \([3,6]\) matches \([1,2]\) by multiplying by 3.
- **Still weak:** The same scalar must work for every component.
- **Current blocker:** I keep saying "both got bigger" instead of checking one shared multiplier.
- **Common mistake:** Comparing component growth separately.
- **Last successful check:** I correctly saw \([3,6]=3[1,2]\).
- **Next best step:** Near-transfer check with a similar vector.
- **Suggested continue prompt:** Continue with one near-transfer check, not from vector basics.

继续。

## Tutor Response

好，我们不从向量基础重讲。你现在真正要稳住的是：**同一个标量必须同时适配每个分量**。

看这个小检查：\([2,6]\) 是不是 \([1,2]\) 的标量倍数？

不要只看“都变大了”。请你分别想：第一个分量需要乘几？第二个分量需要乘几？如果两个数一样，才是同一个标量。

你先判断一下：是，还是不是？
