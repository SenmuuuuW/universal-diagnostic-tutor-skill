# Learner Profile And Task Card Example

## User

我最近在准备考研数学，线代和概率都弱。帮我生成一张长期偏好卡和当前任务卡。

## Expected Tutor Behavior

可以。下面两张卡都是你自己复制保存的，不表示我有隐藏记忆。

```text
Learner Profile Card:
- Preferred language: Chinese
- Current subjects: 考研数学；线性代数；概率论与数理统计
- Learning level: 学过但基础不稳
- Common weak areas: 方法识别、符号意义、错题复盘
- Preferred teaching pace: 一次一个小步骤，检查后再继续
- Exam goal, if any: 考研数学复习
- Platform used: ordinary chat / agent chat
- Notes to preserve: 不要直接给完整答案，先诊断卡点
```

```text
Learning Task Card:
- Task: 修复线代向量与矩阵方程基础
- Topic: \(Ax=b\)、向量平行、标量倍数
- Target outcome: 能识别题型并解释为什么这样做
- Current blocker: 符号和方法线索容易混
- Next action: 做一个 \(u=cv\) 的近迁移检查
- Due date / exam date, if provided: not provided
- Practice needed: 标量倍数、线性组合、方程组设置
- Checkpoint: 先确认同一个标量必须适配所有分量
```

下次新 chat 可以粘贴其中一张卡，然后说：根据这张卡继续，不要从头讲。
