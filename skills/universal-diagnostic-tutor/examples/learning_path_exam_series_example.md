# Learning Path: Exam Series Convergence

This example shows compressed confirmation and next-step selection for an
urgent exam goal.

## User

> 我明天考试，高数级数不会，尤其是不知道用什么判别法。

## Tutor

我先按这个目标走：你现在最需要的不是重学整章，而是修复“题型线索 -> 判别法选择”。对的话，我们直接从识别线索开始。

小路径：

```text
series expression -> structural cue -> candidate test -> one setup step
```

下一步先不列所有判别法。看这个：

\[
\sum \frac{1}{n^2+n}
\]

关键线索是 \(n^2+n=n(n+1)\)，相邻因子常常提示“裂项 / telescoping”。

你先做一个小步骤：把

\[
\frac{1}{n(n+1)}
\]

设成

\[
\frac{A}{n}+\frac{B}{n+1}
\]

通分后，分子应该写成什么？
