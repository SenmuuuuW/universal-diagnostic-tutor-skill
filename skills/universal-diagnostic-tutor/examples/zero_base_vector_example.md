# Zero-Base Vector Example

## User Question

"我是零基础。老师说 vector = endpoint - start point，我完全不懂。"

## Selected Mode

Zero-Base Mode.

## Diagnosis

- **Subject routing:** Linear algebra / geometry foundations.
- **Likely gap:** Vocabulary, notation, and object-type gap.
- **Starting point:** Explain point, vector, and displacement before any
  calculation.

## Expected Pacing Choice

Teach the objects first, use one real-world movement analogy, then stop before
calculation.

## Example Tutor Response

我们先不算，先搞清楚这些东西是什么。

一个 **point（点）** 像地图上的位置：比如你站在 $(2, 1)$。

一个 **vector（向量）** 不是一个位置，而是一段“移动”：从哪里走到哪里、横向走多少、纵向走多少。

所以：

$$
\text{vector} = \text{endpoint} - \text{start point}
$$

意思是：

$$
\text{移动量} = \text{终点位置} - \text{起点位置}
$$

比如你从起点 $(2, 1)$ 走到终点 $(5, 3)$。横向从 $2$ 到 $5$，多了 $3$；纵向从 $1$ 到 $3$，多了 $2$。所以这个向量描述的是“向右 $3$，向上 $2$”。

我先停在这里。小检查：向量是在说“你在哪里”，还是在说“你怎么移动”？

## Stop Point Or Check Question

"向量是在说位置，还是在说移动？"

## Why This Mode Fits The Learner

The learner declared zero-base and is confused by object meaning, so the tutor
starts with point vs vector before doing endpoint-minus-start arithmetic.
