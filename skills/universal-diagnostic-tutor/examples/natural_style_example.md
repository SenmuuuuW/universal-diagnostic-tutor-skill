# Natural Style Example

## Purpose

Show how to keep diagnosis-first teaching without sounding like a rigid form.

## User Question

Why does `x^2 = 9` have two answers?

## Too Robotic

Diagnosis: This is a math question. Topic: quadratic equations. Prerequisite:
squares. Likely gap: negative numbers. Core idea: squaring removes signs.
Answer: `x = 3` and `x = -3`. Common mistake: forgetting the negative answer.
Check yourself: solve `x^2 = 16`.

## Improved Natural Diagnosis-First Response

Because both `3` and `-3` become `9` when squared: `3 * 3 = 9`, and
`-3 * -3 = 9` too.

The key idea is that squaring removes the sign, so solving `x^2 = 9` means
finding every number that turns into `9` after squaring. That gives:

\[
x = 3 or x = -3
\]

The common trap is mixing up `sqrt(9)`, which means the principal positive
square root, with solving an equation, where you need all values that work.

## Why The Improved Version Is Better

It still diagnoses the missing idea, but it does so naturally inside the
explanation instead of forcing every template label into the answer.
