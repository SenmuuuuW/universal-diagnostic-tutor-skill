# Source Specificity Guidelines

Use these guidelines when choosing and citing sources in resource-augmented
tutoring answers.

## Main Rule

Prefer the most specific reliable page that supports the learner's next step.
Use broad homepages as starting points, not as proof that a specific claim,
assignment, or explanation was checked.

## Specificity Ladder

Prefer sources in this order when available:

1. Exact documentation section, standard section, lecture page, assignment page,
   textbook chapter, or course note.
2. Course module, unit page, lab page, or problem-set page.
3. Stable course landing page that clearly routes to the relevant material.
4. Institution or project homepage only when no more specific page is available.

## When To Cite Each Type

- **Course page:** Use for broad learning-path recommendations and prerequisite
  structure.
- **Lecture page or note:** Use when explaining a specific concept.
- **Assignment or lab page:** Use when recommending practice or exam-pattern
  preparation.
- **Documentation page:** Use for current software, API, tool, language, or
  framework behavior.
- **Standard or RFC:** Use for authoritative protocol, format, or system
  requirements.
- **Textbook chapter:** Use for stable foundations and structured follow-up
  reading.

## Avoid Vague Source Lists

Avoid source notes like:

```text
Sources: MIT OCW, Stanford, RFCs, docs.
```

Prefer:

```text
Sources: MIT OCW 18.06 Linear Algebra for matrices as transformations; Stanford
CS144 for transport-layer reliability; RFC 8446 for TLS 1.3 protocol details.
```

## Broad Homepage Use

Use a broad homepage only when:

- It is a stable entry point and exact pages change frequently.
- The learner asked for a course recommendation rather than a specific claim.
- You clearly explain that it is a starting point.
- You do not pretend to have verified a more specific page.

## Learner-Friendly Source Selection

- Choose one to three sources for most answers.
- Separate beginner sources from advanced sources.
- Start with beginner-friendly sources when the learner says "from zero,"
  "basics," or "I am new."
- Escalate to advanced courses, standards, or RFCs only when the learner has
  enough prerequisites or asks for exact technical detail.
- Explain how each source should be used: overview, formal study, practice,
  implementation, or authoritative reference.
