# Exam And Problem Set Sources

Use these sources when a learner wants practice, exam-pattern analysis, or a
study plan built around official or university-style exercises. Do not dump
solutions. Diagnose the tested concept, teach the reusable pattern, then point
the learner to appropriate practice.
