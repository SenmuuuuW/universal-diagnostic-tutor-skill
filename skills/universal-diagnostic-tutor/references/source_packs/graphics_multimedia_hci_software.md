# Graphics, Multimedia, HCI, And Software Source Pack

Use these sources for computer graphics, image processing, multimedia, HCI,
software engineering, VR-adjacent foundations, and human-centered computing.
Prefer university or official sources for concepts; use tutorials for practice
only when their role is clear.

## Sources

### MIT OCW 6.837 Computer Graphics

* Link: https://ocw.mit.edu/courses/6-837-computer-graphics-fall-2012/
* Type: university open course
* Best for: transformations, cameras, rendering, shading, ray tracing, meshes, and animation basics.
* Use when: a learner asks how linear algebra connects to graphics or rendering.
* Strength: Strong university graphics source with assignments.
* Caution: Some toolchains may be dated; use it for concepts more than modern APIs.
* Related subjects: computer graphics, linear algebra, rendering, animation, VR foundations.
* Suggested citation label: MIT OCW 6.837 Computer Graphics

### UC Berkeley CS184 Computer Graphics And Imaging

* Link: https://cs184.eecs.berkeley.edu/sp24
* Type: university open course
* Best for: rasterization, ray tracing, mesh processing, simulation, and imaging concepts.
* Use when: a learner needs a modern university course path for graphics and imaging.
* Strength: Strong conceptual and project-based graphics coverage.
* Caution: Term-specific page; verify exact pages before citing details.
* Related subjects: computer graphics, image processing, simulation, rendering.
* Suggested citation label: UC Berkeley CS184

### OpenGL Wiki Getting Started

* Link: https://www.khronos.org/opengl/wiki/Getting_Started
* Type: official documentation
* Best for: official orientation to OpenGL concepts, ecosystem, and documentation paths.
* Use when: a learner asks where official OpenGL information starts.
* Strength: Khronos-hosted official wiki.
* Caution: Unverified in this environment by command-line check; verify in browser before citing.
* Related subjects: graphics APIs, rendering, GPU programming, computer graphics.
* Suggested citation label: Khronos OpenGL Wiki

### LearnOpenGL

* Link: https://learnopengl.com/
* Type: tutorial
* Best for: practical OpenGL programming, shaders, rendering pipeline, lighting, and textures.
* Use when: a learner needs hands-on OpenGL implementation after concept diagnosis.
* Strength: Widely used practical tutorial with clear examples.
* Caution: Command-line check returned a nonstandard access response in this environment. Treat as tutorial-level, verify before citing, and pair with official docs or university sources for conceptual claims.
* Related subjects: OpenGL, graphics programming, shaders, rendering.
* Suggested citation label: LearnOpenGL

### Nielsen Norman Group Usability Heuristics

* Link: https://www.nngroup.com/articles/ten-usability-heuristics/
* Type: authoritative technical reference
* Best for: HCI usability principles, interface evaluation, and design critique vocabulary.
* Use when: a learner asks how to analyze or improve an interface.
* Strength: Widely cited HCI guidance with practical evaluation language.
* Caution: It is not a substitute for user research or domain-specific accessibility checks.
* Related subjects: HCI, UX, software design, interface evaluation.
* Suggested citation label: Nielsen Norman Group Usability Heuristics

### Software Engineering At Google
