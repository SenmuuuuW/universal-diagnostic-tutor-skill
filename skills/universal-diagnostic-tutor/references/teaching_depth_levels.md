# Teaching Depth Levels

Choose the least depth that still produces understanding.

## Level 1: Answer + One-Line Reason

Use when:

- The user asks for a quick answer.
- The task is a confirmation or simple fact check.
- The learner likely already understands the method.

Shape:

1. Answer.
2. One-sentence reason.

## Level 2: Brief Explanation

Use when:

- The user asks "why" but the concept is narrow.
- A short correction is enough.
- The question is low stakes.

Shape:

1. Diagnosis in one sentence.
2. Short explanation.
3. Final answer or takeaway.

## Level 3: Standard Teacher-Style Explanation

Use as the default.

Shape:

1. Diagnosis: subject, topic, prerequisite.
2. Core idea.
3. Step-by-step explanation.
4. Final answer or conclusion.
5. Similar-problem rule.
6. Short check question.

## Level 4: Foundation-First Full Explanation

Use when:

- The learner says they are confused.
- The problem depends on missing prerequisites.
- The topic is abstract or cumulative.

Shape:

1. Diagnose what must be understood first.
2. Teach the prerequisite idea in plain language.
3. Connect the prerequisite to the current problem.
4. Solve or explain step by step.
5. Name common mistakes.
6. Give practice.

## Level 5: Knowledge-System Explanation

Use when:

- The user wants deep understanding.
- The topic belongs to a larger conceptual system.
- The user is preparing for an exam, interview, paper, or project.

Shape:

1. Map the topic inside the broader knowledge system.
2. Explain prerequisites and vocabulary.
3. Teach the core model or framework.
4. Work through examples.
5. Show real-world applications or disciplinary significance.
6. Identify common mistakes and transfer patterns.
7. Give practice questions.

## Adapting Depth

Reduce depth when the user asks for speed. Increase depth when the user asks
"why", "from the basics", "I am lost", "teach me", or "how do I know when to
use this?"

When depth increases because the learner is stuck, do not only add more detail.
Change the teaching move: repair an earlier prerequisite, switch
representation, use a smaller example, or add a practice rung.
