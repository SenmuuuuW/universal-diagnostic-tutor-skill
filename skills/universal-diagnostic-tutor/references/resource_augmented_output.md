# Resource-Augmented Output Formats

Use these formats when external learning resources are used or requested.
Adapt them naturally; do not force every section into every answer.

## Concept Explanation With Sources

```text
Diagnosis: This is [subject] about [topic]. The likely prerequisite gap is
[gap].

Resource note: I used [source types/names] because [reason].

Core idea: [plain-language explanation]

Formal view: [definition, formula, or model]

Why it matters: [application or later-course connection]

Common mistake: [pitfall]

Practice direction: [what to try next]

Sources: [source list]
```

## Problem Solving With Sources

```text
Diagnosis: This problem tests [concept] and depends on [prerequisite].

Resource note: [problem set/course/source] frames this as [pattern].

Solution path:
1. [step with reason]
2. [step with reason]
3. [step with reason]

Answer: [final result]

How to recognize similar problems: [transfer rule]

Sources: [source list]
```

## Exam-Pattern Analysis

```text
Tested concept: [concept]
Prerequisites: [prerequisites]
Pattern: [recurring question form]
Trap: [common wrong path]
Method: [why the correct method works]
Practice next: [practice direction]
Sources: [source list]
```

## Learning Path Recommendation

```text
Diagnosis: Your goal is [goal], and the prerequisite chain is [chain].

Recommended path:
1. [foundation]
2. [core concept]
3. [implementation or practice]
4. [advanced connection]

Practice plan: [short plan]

Sources: [source list]
```

## Source-Limited Answer

Use when search is unavailable or blocked.

```text
I can explain this from foundations, but I could not verify external resources
in this environment.

Diagnosis: [subject/topic/gap]

Foundation explanation: [teaching]

What to verify later: [source checklist]
```

## Sources Found But Not Enough

Use when retrieved sources are relevant but incomplete or low confidence.

```text
I found [source types], but they are not enough to fully verify [claim].

What they support: [supported points]
What remains uncertain: [limits]
Teaching from foundations: [careful explanation]
Sources checked: [source list]
```

## Foundations-Only With Search Unavailable

Use when the learner needs help now but no external search is possible.

```text
External search is unavailable, so I will separate general explanation from
source-backed claims.

General explanation: [teach from internal knowledge]

Suggested resources to look for later: [source categories, not invented links]
```
