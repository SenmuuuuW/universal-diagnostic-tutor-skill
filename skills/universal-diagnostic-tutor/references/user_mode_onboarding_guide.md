# User Mode Onboarding Guide

Use this guide when documentation, examples, or a first tutoring turn should
invite the learner to choose a teaching mode. Keep it short and friendly.

## Chinese User Guide

你可以先告诉导师你希望用哪种模式学习：

- 🌱 **零基础模式**：适合完全没学过，需要从概念、符号、例子讲起。
- 📘 **普通模式**：适合学过一点，但不会做题或不会判断方法。
- 🚀 **进阶模式**：适合有基础，想要更快、更严谨、更深入。
- **Auto Mode**：如果你不选择，导师会根据你的问题、用词、错误和信心自动判断；不确定时会问一个很短的问题。

## Copyable Prompt Examples

```text
我是零基础，请从概念和符号开始教我这道题，不要直接给答案。
```

```text
我学过一点，但不会判断用什么方法，请用普通模式一步一步教我。
```

```text
我有基础，请用进阶模式讲核心思路、证明和易错点。
```

```text
先按普通模式讲，如果发现我基础不够，再自动降到零基础模式。
```

```text
只讲到第一个关键步骤就停，等我回答后再继续。
```

## Tutor Onboarding Phrases

Use sparingly:

- "你可以选零基础、普通或进阶模式；如果不选，我会自动判断。"
- "我先按普通模式讲；如果你觉得太快，我会降到零基础。"
- "你已经有基础，我会省略定义，直接讲证明关键点。"

Avoid turning every answer into a mode menu. Offer modes when the learner is
new, unclear, or explicitly choosing how they want to learn.
