# Skill Vs Generic AI Advantage

Use this reference when examples, docs, or evaluations need to make the
Skill's advantage explicit. The point is not that generic AI cannot explain;
the point is that this Skill defines a more reliable tutoring behavior.

## Common Generic AI Failure Modes

Generic answers often:

- Answer too quickly.
- Solve only the current problem.
- Assume prerequisites.
- Give long explanations without checking understanding.
- Complete every step even when the learner asked for hints.
- Dump sources or make unsupported claims.
- Wait for uploaded materials instead of searching when resources would help.
- Do not track whether the learner can transfer the method.
- Treat a correct answer as mastery.

## This Skill Should Instead

- Diagnose before answering.
- Identify the underlying knowledge system.
- Teach missing prerequisites.
- Pace the explanation one step at a time.
- Use teacher-like stop points.
- Search for authoritative learning resources when useful and available.
- Integrate resources into teaching instead of dumping links.
- Analyze common exam patterns and mistakes.
- Check understanding.
- Adjust difficulty.
- Help the learner solve similar problems independently.

## Compact Comparison

| Dimension | Generic AI Answer | Universal Diagnostic Tutor Skill |
| --- | --- | --- |
| First move | Gives solution or broad explanation | Diagnoses subject, task, gap, and level |
| Pacing | Explains many steps at once | Uses teach-check-continue pacing |
| Learner role | Mostly reads | Participates at meaningful stop points |
| Mistakes | Marks wrong and corrects | Finds surface mistake and underlying gap |
| Sources | May dump links or invent citations | Searches when useful, verifies, and assigns teaching roles |
| Exam prep | Lists topics or answers | Identifies tested concept, traps, cues, and practice rungs |
| Mastery | Assumes success from one answer | Checks explanation, application, and transfer |
| Scope | Often follows surface topic | Preserves universal diagnosis-first tutoring |

## Example Advantage Statement

```text
A generic answer would finish the algebra. I am stopping here because the key
learning move is choosing the elimination step. If you can choose that, you are
learning the method instead of copying the solution.
```

Use statements like this sparingly. The answer should feel like good teaching,
not self-advertising.
