# Goal Clarifier Protocol

Use this protocol when the learner gives a broad or underspecified learning
goal, such as "我想学机器学习", "我线代很差", "我想准备考试", or "I want to
learn algorithms."

The purpose is to learn enough about the goal to choose a useful first path.
It is not a survey, intake form, or full curriculum interview.

## What To Clarify

Identify only the pieces needed for the next step:

- **Learning target:** concept, problem type, course area, exam, project, or
  broad subject.
- **Motivation:** exam, homework, project, interview, research, self-study, or
  general curiosity.
- **Current level:** zero-base, some exposure, advanced, or unknown.
- **Time constraint:** urgent, short-term, semester-long, or open-ended.
- **Desired output:** explanation, problem-solving ability, practice, plan,
  resource direction, project ability, or review.
- **Subject scope:** one concept, one unit, one course module, or a broader
  field.
- **Context:** exam, project, general learning, mistake repair, or foundation
  repair.

## Question Budget

- Ask 1-3 focused questions.
- Ask fewer questions when the learner is urgent, overwhelmed, or already
  specific.
- If the user gives enough context, do not over-question; proceed to goal
  confirmation or the first teaching step.
- Prefer choice-based questions over open-ended interviews.

## Common Goal Types

| Goal type | Likely next move |
| --- | --- |
| Understand a concept | Clarify current level and desired depth |
| Solve a problem | Diagnose subject, subtopic, and first blocker |
| Prepare for an exam | Clarify exam/course and weakest topic |
| Build a project | Clarify project output and prerequisite gaps |
| Learn a course or field | Clarify motivation, current level, and scope |
| Repair weak foundations | Find earliest unstable prerequisite |
| Review mistakes | Ask for one representative mistake or pattern |
| Create a study plan | Clarify current state, deadline, and goal |

## Example Tutor Moves

### Machine Learning Broad Goal

User:

> 我想学机器学习。

Tutor:

> 这个目标有点大，我先帮你缩小一下：你是为了做项目、准备考试、看懂基础模型，还是系统入门？你现在数学和 Python 大概到什么程度？

### Weak Linear Algebra

User:

> 我线代很差，想补一下。

Tutor:

> 你是为了课程考试、考研数学，还是为了后面学机器学习？你现在最卡的是向量、矩阵、行列式，还是特征值？

### Urgent Series Exam

User:

> 我明天考试，高数级数不会。

Tutor:

> 那我们不做长问卷。你现在主要卡在判别法选择，还是每个判别法本身不会用？

## Mistakes To Avoid

- Asking a long questionnaire before helping.
- Treating a broad field goal as permission to generate a huge curriculum map.
- Starting advanced content before the goal and level are clear.
- Ignoring urgent exam context.
- Assuming that a project learner and an exam learner need the same first step.
