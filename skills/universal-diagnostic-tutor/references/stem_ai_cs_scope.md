# STEM / AI-CS Scope

This is a primary-use reference pack for university-level science and
engineering learning, especially AI, computer science, mathematics, physics,
and related foundations.

It does not replace the universal tutor identity. Continue supporting
humanities, writing, law/civics, economics, language learning, health education,
finance education, exam prep, and general learning.

## Math Foundations

- Calculus: limits, derivatives, integrals, series, multivariable ideas, and
  optimization intuition.
- Linear algebra: vectors, matrices, transformations, eigenvalues, vector
  spaces, orthogonality, and numerical meaning.
- Discrete math: logic, sets, relations, graphs, counting, induction, and proof.
- Probability and statistics: random variables, distributions, estimation,
  hypothesis testing, and stochastic processes.
- Complex functions, numerical analysis, number theory, and mathematical
  experiments.

## Physics And Electronics Foundations

- Mechanics, electromagnetism, waves, optics, and experiments.
- Circuit models, digital logic, electronics fundamentals, and measurement.
- Signal processing: time/frequency views, transforms, filtering, sampling, and
  engineering interpretation.
- Control and systems: models, stability, feedback, and applications.

## Programming Foundations

- Programming fundamentals and practice.
- Object-oriented programming.
- Assembly language and low-level execution.
- Software engineering, testing, design, and maintainability.

## Computer Systems

- Data structures and algorithmic complexity.
- Computer organization and architecture.
- Operating systems: processes, memory, files, scheduling, concurrency, and
  virtualization.
- High-performance computing, storage, and systems design.

## Algorithms And Theory

- Formal languages and automata.
- Compiler principles.
- Cryptography and security foundations.
- Complexity reasoning and proof patterns.

## AI / ML Foundations

- Introduction to AI: search, planning, representation, and reasoning.
- Machine learning: linear models, optimization, probability, generalization,
  evaluation, and implementation.
- Neural networks: layers, activations, loss functions, backpropagation,
  embeddings, and representation learning.
- Pattern recognition, data mining, search engines, image processing, and
  multimedia.

## Data, Databases, Search, Networks, And Security

- Databases: relational models, queries, indexing, transactions, and project
  design.
- Search engines: indexing, ranking, retrieval, and evaluation.
- Networks: layering, protocols, reliability, congestion, routing, and real
  applications.
- Security: threat models, cryptography, network security, and system risks.

## Graphics, HCI, VR, And Multimedia

- Computer graphics fundamentals, transformations, rendering pipelines, and
  geometry.
- Human-computer interaction, usability, cognitive load, and design evaluation.
- Virtual reality, multimedia technology, and digital image processing.

## How The Subjects Connect

Use prerequisite chains to orient the learner:

- Linear algebra + calculus + probability -> machine learning and neural
  networks.
- Programming + data structures + algorithms -> software systems and AI
  implementation.
- Digital logic + organization + architecture -> operating systems and high
  performance computing.
- Networks + operating systems + security -> distributed systems and secure
  applications.
- Physics + electronics + signals -> image processing, controls, hardware, and
  sensing.
- Databases + search + data mining -> information retrieval and applied AI.

When teaching, connect the immediate topic to earlier prerequisites and later
courses so the learner sees the knowledge system rather than isolated facts.
