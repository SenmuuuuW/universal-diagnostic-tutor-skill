# Source Note Checklist

Use this checklist before citing or listing sources in a resource-augmented
tutoring answer.

## Before Citing

- Did I use sources appropriate for the learner's topic and level?
- Are official, university, open textbook, standards, or authoritative sources
  preferred where possible?
- Did I avoid fake, invented, unverifiable, or hallucinated citations?
- Did I avoid answer-only sites and sources that skip reasoning?
- Did I cite specific course pages, lecture pages, documentation sections,
  problem-set pages, or textbook sections when possible instead of vague
  homepages?
- Did I cite a specific page when possible?
- Am I citing a broad homepage only because it is the best available starting
  point?
- Did I avoid overwhelming the learner with too many sources?
- Did I separate beginner sources from advanced sources?
- Did I flag term-specific or potentially stale course pages?
- Did I distinguish source-backed points from my own explanation?
- Did I explain how the source helps the learner continue studying?
- Did I mention if source access was unavailable or only partially verified?
- Did I keep source notes short enough that teaching remains the center?

## Good Source Note

```text
Source note: I would use MIT OCW 18.06 for the linear algebra foundation and
Stanford CS224N for the embedding/NLP connection. MIT 18.06 supports the matrix
and vector-space part; CS224N supports the representation-learning application.
```

## Weak Source Note

```text
Sources: MIT, Stanford, YouTube, some ML docs.
```

This is weak because it is vague, does not say why the sources fit the
learner's gap, and does not separate formal sources from intuition sources.
