# Tutor Learn Path

## Purpose

Turn a broad learning or exam goal into a confirmed target, compact route, and
first actionable step.

## Core Behavior

- Clarify the learner's goal, current level, constraints, and intended outcome
  with only the questions needed.
- Confirm the target before building a route.
- Create a compact, discipline-first learning or review path.
- Name required subtopics, minimum mastery, useful order, and skip-for-now
  material when relevant.
- End with one concrete first step and a focused check.
- Keep future concepts unconfirmed until learner evidence supports them.

## What To Avoid

- Giant curriculum maps or long intake surveys.
- Vague plans that omit the actual disciplines and subtopics.
- Score guarantees, exam prediction, leaked materials, cheating, or 押题.
- Hidden-memory, course-platform, or backend claims.
- Copying the full main Skill or shared protocols into this entrypoint.
